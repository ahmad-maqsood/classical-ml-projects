# Classical ML Projects
 
A collection of machine learning projects built using classical ML algorithms. Each project covers the full pipeline: data cleaning, exploratory data analysis, visualization, model training, and prediction.
 
## Tools & Libraries
 
- Python, Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn
- Joblib
 
---
 
## Structure
 
Each project almost follows the same structure:
 
```
project-name/
│
├── data/               # Raw and cleaned datasets
├── model/              # Saved model and encoders
├── 1_data_analysis.ipynb
├── 2_data_visualization.ipynb
├── 3_model_training.ipynb
└── 4_predict_using_model.py
```
